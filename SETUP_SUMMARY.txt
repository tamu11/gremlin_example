## 🎯 What It Does

This system models relationships between:
- **People** (employees with expertise)
- **Documents** (research papers, white papers)
- **Knowledge Areas** (topics like Machine Learning, Cloud Computing)

It creates relationships: Person → AUTHORED → Document → COVERS → KnowledgeArea

Then queries can traverse these relationships to find experts for specific topics!

## 🚀 How to Run

### Windows (PowerShell - Recommended)

```powershell
# Start database
./start_database.ps1

# Load data (wait 30-60 seconds)
python scripts/load_sample_data.py

# Query for experts
python scripts/query_experts.py

# Stop when done
./stop_database.ps1
```

### Linux/macOS (Bash)

```bash
# Start database
./scripts/start_database.sh

# Load data (wait 30-60 seconds)
python scripts/load_sample_data.py

# Query for experts
python scripts/query_experts.py

# Stop when done
./scripts/stop_database.sh
```

### All Platforms (Docker Directly)

```bash
# Start JanusGraph
docker run -d --name janusgraph -p 8182:8182 -v $(pwd)/conf:/janusgraph/conf janusgraph/janusgraph

# Wait 30 seconds, then:
python scripts/load_sample_data.py
python scripts/query_experts.py

# When done:
docker stop janusgraph
```
