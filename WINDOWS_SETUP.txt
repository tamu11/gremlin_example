## 🚀 Windows Quick Start

### Using PowerShell (Recommended)

1. **Start the database**
   - Right-click `start_database.ps1`
   - Select "Run with PowerShell"
   - This will start the JanusGraph container

2. **Load sample data** (wait 30-60 seconds after starting)
   - Open PowerShell or CMD
   - Navigate to the directory: `cd D:\Workspace\gremlin_example`
   - Run: `python scripts/load_sample_data.py`

3. **Query for experts**
   - Run: `python scripts/query_experts.py`
   - You'll see results showing experts by topic

4. **Stop when done**
   - Right-click `stop_database.ps1`
   - Select "Run with PowerShell"

### Using Docker Directly

```powershell
# Start JanusGraph
docker run -d --name janusgraph -p 8182:8182 -v ${env:PWD}\conf:/janusgraph/conf janusgraph/janusgraph

# Wait 30 seconds, then:
python scripts/load_sample_data.py
python scripts/query_experts.py

# When done:
docker stop janusgraph
```

## 💡 Troubleshooting Tips

1. **Docker Desktop must be running** - Make sure it's started
2. **Wait for initialization** - After starting, wait 30-60 seconds before loading data
3. **Port conflicts** - Check if port 8182 is available
4. **Permission issues** - Run PowerShell as Administrator if needed
